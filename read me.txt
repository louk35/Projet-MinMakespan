Dans le cas où PyCharm n'est pas installé, pas de problèmes!

1/ Ouvrez IDLE(python)
2/ En haut à gauche, aller dans file
3/ Open (selectionner le fichier minmakespan.py)
** Une autre fenêtre s'ouvre **
4/ Sur celle-ci, en haut cliquez sur Run
5/ Run module ( puis le programme se lance )
